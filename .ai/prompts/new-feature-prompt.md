# New Feature Prompt

Implement a new Habit Compass feature using spec-driven development.

Required steps:

1. Read `AGENTS.md`.
2. Read `docs/engineering/ai-code-task-guardrails.md`.
3. Read project context and product principles.
4. Read or create the feature spec.
5. Confirm acceptance criteria.
6. Implement the smallest useful slice.
7. Add tests and i18n.
8. Run verification.
