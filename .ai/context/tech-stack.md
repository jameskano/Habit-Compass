# Tech Stack

- React with TypeScript and Vite.
- Tailwind CSS with class-based dark mode and a shadcn/ui-ready structure.
- TanStack Router for routing.
- TanStack Query for server state.
- Zustand for local UI/app state.
- React Hook Form, Zod, and `@hookform/resolvers` for forms.
- Supabase Auth, Postgres, RLS, and Edge Functions later.
- Supabase JS client configured with environment placeholders.
- react-intl for localization.
- date-fns for date logic.
- Recharts for lightweight visualizations.
- Vitest, React Testing Library, and Playwright for tests.
- ESLint and Prettier for static checks and formatting.
- Sentry-ready integration with safe missing-DSN behavior.
- Capacitor-ready dependencies and documentation, without native platform initialization.
- pnpm for package management.
