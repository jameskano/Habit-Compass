# ADR 0001: Tech Stack

## Decision

Use React, TypeScript, Vite, Tailwind CSS, TanStack Router, TanStack Query, Zustand, React Hook Form, Zod, Supabase, react-intl, date-fns, Recharts, Vitest, React Testing Library, Playwright, Sentry, Capacitor, and pnpm.

## Rationale

The stack supports a fast web MVP, future Supabase persistence, strong typing, mobile readiness, and reliable testing without committing to native platforms yet.
