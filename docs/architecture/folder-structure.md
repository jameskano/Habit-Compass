# Folder Structure

The repository is organized for spec-driven product work and future mobile readiness.

- `.ai`: AI context, workflows, agents, skills, and prompts.
- `specs`: product and behavior specs.
- `docs`: architecture, testing, product, database, and ADR documentation.
- `src/app`: app shell.
- `src/domain`: domain model and rules.
- `src/features`: feature surfaces.
- `src/integrations`: external service boundaries.
- `src/test`: fixtures, harnesses, and utilities.
