# Feature Flags

Use feature flags for optional or future depth:

- Future Mood and reflections.
- Weekly planning depth.
- Suggestions.
- Future AI.
- Future calendar integration.
- Future subscriptions.

Flags should default to the simplest stable experience.

The final MVP Settings screen does not expose optional-depth feature toggles as its primary content.
Feature-toggle state may remain for onboarding, compatibility, and future feature gates, but Settings
uses the information architecture in [settings-spec.md](../../specs/mvp/settings-spec.md).
