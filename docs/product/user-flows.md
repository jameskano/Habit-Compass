# User Flows

Initial flows:

- Open Today and complete an item.
- Add a habit, task, or recurrent task.
- Review and manages the week.
- Optionally add mood or reflection.
- Manage active and archived items, with explicit confirmed deletion when needed.
- Change theme in settings.
- Manage Settings preferences: language, theme, and week start.
- Export data from Data and privacy.
- Submit feedback or rate the app from Support and feedback.
- Sign out locally from the current device/session.
- Request account deletion with a seven-day pending-deletion period.
